# file-ace
Organizing your digital life, one file at a time.

## Description
Tame digital chaos with file-ace, a Python-powered file sorter. This handy tool helps you keep your files in check, saving you time and reducing stress. With file-ace, you can easily categorize and move files to their designated directories, keeping your digital life organized and clutter-free.

## Features
* **Customizable file categorization**: Easily define your own file categories and sorting rules to fit your specific needs.
* **Automated file sorting**: file-ace takes care of moving files to their designated directories, saving you time and effort.
* **Support for multiple file types**: file-ace can handle a wide range of file types, from documents and images to videos and more.
* **Lightweight and efficient**: Written in Python, file-ace is designed to be fast and resource-friendly.

## Installation and Usage
To get started with file-ace, follow these simple steps:

1. **Install dependencies**: Run `pip install -r requirements.txt` to install the required packages.
2. **Run file-ace**: Execute `python main.py` to start the file sorter.
